# Zitadel Docker Compose
- [Docs](https://docs.zitadel.com/)

## Required
- Docker
- Docker Compose Plugin

## Installation
- Configure `install.yaml`
- Configure `config.yaml`
- `docker compose up -d`
- Goto `your.domain/ui/console`
- Login as `root`@zitadel.`your.domain` (Password is `Password1!`)
